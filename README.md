## 例題
 - 例題等のダウンロードは、次のいずれかの方法でどうぞ
   - ページ右上にある緑色のボタン［Code］をクリックして、［Download ZIP］をクリック（初心者はこちらのほうが簡単かと思われます）
   - 適当なフォルダを作り、そこに移動し、コマンドラインで次を実行（詳しくは本の10章を参照してください）
     <pre>
      git clone https://github.com/urupat/se4ds.git
     </pre>
